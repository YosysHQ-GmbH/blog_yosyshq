The April release of Tabby CAD Suite is now available for download. Notable changes in this version include:

* Added initial synthesis support for AnalogDevices FPGAs.
* Removed rarely-used options from ABC/ABC9 passes.
* Calls to "abc -g AND -fast" to map logic to AND-Inverter Graph form should be replaced with "aigmap".
* Please check your custom scripts and update accordingly.

In other YosysHQ news:

* CSO Matt Venn will be attending DATE for a panel discussion on Wednesday 22nd April. If you’re attending, make sure to say hi!
* Applications for the Tiny Tapeout demoscene competition are now open! Learn more and apply [here](https://tinytapeout.com/competitions/demoscene-ttsky26a-announce/).
