The September release of Tabby CAD Suite is now available for download. Notable changes in this version include:

* The SBY option `skip` now works in cover mode.
* Added the option <code>[abstract -state -initstates N](https://yosyshq.readthedocs.io/projects/yosys/en/latest/cmd/index_passes_cmds.html#abstract-replace-signals-with-abstract-values-during-formal-verification)</code> that allows activating initial value abstraction on a register for the first `N` steps in a trace.
